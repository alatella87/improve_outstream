This ad container is based on the Google IMA SDK for HTML5 V3, which usually acts as a point of reference for displaying linear, non-linear, and companion ads in videos and games.

About this attempt

This testing instance has all references to the otherwise needed video content disabled, making it an easy solution to implement as an outstream/inRead player. Specifically, the main video reference in the DOM and Google's videoContent reference in ads.js are substituted to convert the IMA SDK in an outstream solution.
